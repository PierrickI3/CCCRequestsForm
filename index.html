Welcome to the Competence Center Request Form.

This form can be used to request assistance on a variety of topics across different product types.

Fill out the form as required. Some fields are automatically completed using your Genesys Cloud details. Press Submit once all the required fields have been completed.

  1. Region
      Enter the region you are located in.
  2. Territory
      Enter the territory you are located in.
  3. Segment
      Select a Sales segment
  4. Product
      Select the product name
  5. Request Type
      Enter the request type you require. Please note that not all request types are available in every region.
  6. Requester Phone #
      Enter the phone number where you can be contacted. Be sure to include your country code, for example +447557123456.
  7. Need Completed By
      Enter a date you require the work completed. Please note, this is not a guarantee and will be discussed with the Competence Centre Program Manager when they contact you.
  8. Description
      Enter a brief summary of the work required. Your Competence Centre Program Manager will contact you to discuss the request further.
  9. Partner and/or Customer Name
      If applicable, enter the Partner and/or Customer name the request is related to.
  10. SFDC URL (Account or Opportunity)
      If applicable, provide the link to the account or opportunity in Salesforce. This will provide the Competence Centre Program Manager with more background and will help with your resource allocation.
